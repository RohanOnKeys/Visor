Primary goal
Turn raw webpage DOM into safe, structured, agent-ready context.
MVP boundary
Chrome extension, local-only compiler, JSON/Markdown export, privacy redaction, tests.
1. Roadmap strategy
Build the compiler core before expanding UI, cloud, or agent integrations. Each roadmap phase should increase
correctness, safety, and integration value without breaking the local-first principle.

7. Kill criteria
•
Do not continue UI expansion if extraction quality is poor.
•
Do not add backend if local compiler is not reliable.
•
Do not publish publicly if redaction and permission review are incomplete.
•
Do not claim autonomous agent capability before action safety is designed.
