Primary goal
Turn raw webpage DOM into safe, structured, agent-ready context.
MVP boundary
Chrome extension, local-only compiler, JSON/Markdown export, privacy redaction, tests.
1. Build mission
Visor converts the current webpage into a structured context object that an AI agent can use safely. The product is not a
scraper dump. It is a compiler pipeline: raw DOM -> page snapshot -> normalized content -> privacy-checked context ->
agent-ready export.
2. MVP non-negotiables
•
Chrome-first, Manifest V3 extension.
•
TypeScript-first codebase with strict types.
•
Local-first processing; no backend or external AI calls in MVP.
•
Minimal permissions: prefer activeTab plus scripting/storage where possible; host permissions only when needed.
•
Every extracted webpage instruction is treated as untrusted data, never as developer/system instruction.
•
User must preview context before export or external sharing.
•
Implementation must include tests for extraction, compiler, privacy redaction, and export formatting.

4. MVP feature list
•
Compile the active tab into a clean AgentContext JSON object.
•
Extract visible text, headings, links, buttons, forms, tables, metadata, and code blocks.
•
Remove obvious noise: hidden nodes, scripts, styles, boilerplate, cookie banners, repeated nav/footer text.
•
Estimate token usage and support compact, detailed, action, RAG, and debug modes.
•
Redact obvious sensitive content and warn on risky page classes.
•
Preview before export and copy JSON, Markdown, or prompt block output.
•
Run local-first with no external AI calls in MVP.

7. First AI coding-agent prompt
You are building a production-quality Chrome Manifest V3 extension called Visor DOM-to-Agent
  Context Compiler.
Goal: Convert the current webpage DOM into clean, structured, privacy-safe, agent-ready
  context.
MVP only:
- TypeScript, React, Vite, Manifest V3.
- Content script extracts URL, title, meta description, headings, visible text, links,
  buttons, forms, tables, and code blocks.
- Service worker coordinates messages.
- Compiler removes noise, deduplicates text, estimates tokens, and outputs AgentContext
  JSON.
- Privacy redactor detects emails, phones, password fields, API-key-like strings, and JWT-
  like strings.
- Popup has Compile Page, JSON preview, Copy JSON, and Copy Markdown.
- No backend. No external AI calls. Minimal permissions. Local-first.
Return folder structure, complete code, build instructions, testing instructions, and
  architecture explanation.
