1. Extraction goal
The extractor creates an intermediate PageSnapshot from the active page. It must preserve content needed for an AI agent
while removing or labeling material that is hidden, duplicated, irrelevant, unsafe, or non-semantic.

5. Visibility heuristic
function isProbablyVisible(el: Element): boolean {
  const style = window.getComputedStyle(el);
  const rect = el.getBoundingClientRect();
  if (style.display === 'none' || style.visibility === 'hidden') return false;
  if (el.hasAttribute('hidden') || el.getAttribute('aria-hidden') === 'true') return false;
  if (rect.width === 0 && rect.height === 0 && !isSemanticInput(el)) return false;
  return true;
}
6. PageSnapshot schema
type PageSnapshot = {
  schemaVersion: 'page_snapshot.v1';
  source: {
    url: string;
    canonicalUrl?: string;
    title: string;
    capturedAt: string;
    language?: string;
  };
  metadata: Record<string, string>;
  headings: HeadingBlock[];
  textBlocks: TextBlock[];
  links: LinkBlock[];
  actions: ActionBlock[];
  forms: FormBlock[];
  tables: TableBlock[];
  media: MediaBlock[];
  codeBlocks: CodeBlock[];
  stats: DOMStats;
  warnings: ExtractionWarning[];
};
7. Edge cases
Case
Expected behavior
Shadow DOM
Traverse open shadow roots when safe; report closed shadow roots as inaccessible.
Iframes
Same-origin iframe may be inspected; cross-origin iframe must be reported as inaccessible.
Infinite scroll
MVP extracts currently loaded DOM only; do not auto-scroll.
Dynamic React apps
Extract visible rendered DOM; do not infer hidden application state.
Very large DOM
Apply node cap, block cap, and partial snapshot warning.
Canvas-only page
Return metadata and warning that semantic DOM content is limited.
