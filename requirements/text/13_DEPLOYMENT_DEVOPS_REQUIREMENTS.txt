Primary goal
Turn raw webpage DOM into safe, structured, agent-ready context.
MVP boundary
Chrome extension, local-only compiler, JSON/Markdown export, privacy redaction, tests.
1. Deployment position
For MVP, deployment means a local unpacked Chrome extension build that developers can install from the dist folder.
Store publishing, CI, signing, and backend deployments are future steps after compiler quality and security are proven.
2. Local development commands
npm install
npm run dev
npm run typecheck
npm run test
npm run build
Chrome manual install:
1. Open chrome://extensions
2. Enable Developer mode
3. Click Load unpacked
4. Select the dist/ folder
5. Pin Visor and test on fixture pages
